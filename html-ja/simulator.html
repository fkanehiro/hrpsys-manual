シミュレータ¶

hrpsys-baseは以下の物理シミュレータと統合されたシステム構築ツールを提供します。

hrpsys-simulatorコマンド¶

hrpsys-simulatorコマンドはシミュレーションをロードし実行します。以下のように使います:

$ hrpsys-simulator [project file] [options]

**プロジェクトファイル**はシステムの設定を定義するXML形式のファイルです（あとで説明します）。

hrpsys-simulatorコマンドには以下のオプションがあります:

-nodisplay¶

ヘッドレスモード

-realtime¶

実時間への同期

-usebbox¶

干渉チェックに外接四角形を使う

-endless¶

エンドレスモード

-showsensors¶

画面上でセンサを視覚化する

-size [pixels]¶

ウィンドウの大きさを設定する

-no-default-lights¶

環境光を無効にする

-max-edge-length [value]¶

ボリゴンの最大長を設定する（この最大長より大きなポリゴンは自動で分割され陰影などの表示品質を改善することができます）

-max-log-length [value]¶

ログの最大長を設定する

-exit-on-finish¶

シミュレーションが終了したらプログラムを終了する

-record¶

シミュレーションを動画で保存する

-bg [r] [g] [b]¶

背景色を設定する

-h --help¶

ヘルプを表示する

hrpsys-simulator-pythonコマンド¶

hrpsys-simulator-pythonコマンドはシミュレータをロードし実行します。以下のように実行します:

$ hrpsys-simulator-python [project file] [python script] [options]

project file はシステムの構成を定義するXML形式のファイルです（あとで説明します）。python script はシミュレーション開始時に実行されるPythonスクリプトです。

hrpsys-simulator-pythonコマンドはhrpsys-simulatorと同じオプションを使うことができます。

hrpsys-simulator-pythonコマンドは、システムとシミュレーションの初期化が終わったあと対話的に操作できるプロンプト画面に移行します。プロンプト画面からPythonコマンドを入力することでシミュレーションをインタラクティブに操作することができます。

hrpsys-simulator-jythonコマンド¶

hrpsys-simulator-jythonコマンドはシミュレータをロードし実行します。以下のように実行します:

$ hrpsys-simulator-jython [project file] [jython script] [options]

project file はシステムの構成を定義するXML形式のファイルです（あとで説明します）。python script はシミュレーション開始時に実行されるjythonスクリプトです。

hrpsys-simulator-jythonコマンドはhrpsys-simulatorと同じオプションを使うことができます。

hrpsys-simulator-jythonコマンドは、システムとシミュレーションの初期化が終わったあと対話的に操作できるプロンプト画面に移行します。プロンプト画面からjythonコマンドを入力することでシミュレーションをインタラクティブに操作することができます。

XMLプロジェクトファイル¶

XML形式のプロジェクトファイルには複数の設定項目があります。各設定項目は**classアトリビュート**によって識別されます。hrpsys-simulatorには現在以下の設定タイプがあります。

com.generalrobotix.ui.item.GrxRTSItem

RTコンポーネントの初期化と接続を設定する

com.generalrobotix.ui.item.GrxModelItem

ロボットモデルのロードと初期姿勢を設定する

com.generalrobotix.ui.item.GrxCollisionPairItem

2つのオブジェクトの間の干渉を設定する。

com.generalrobotix.ui.item.GrxWorldStateItem

物理シミュレーションのパラメータを設定する

com.generalrobotix.ui.item.GrxPythonScriptItem

シミュレーションの裏で動作するpythonスクリプトを設定する

com.generalrobotix.ui.view.Grx3DView

シミュレーション画面の初期のカメラ位置と視覚化を設定する

com.generalrobotix.ui.grxui.GrxUIPerspectiveFactoryなど、その他の設定項目もありますが、それらはeclipseベースの旧OpenHRPシミュレータのための設定項目で互換性のために存在します。hrpsys-simulatorでは無視してください。

GrxRTSItem¶

**com.generalrobotix.ui.item.GrxRTSItem**設定項目では、”.period”が各RTコンポーネントの実行周期を設定します。”.factory”はコンポーネントのインスタンスを作成するのに用いられる共有ライブラリを設定します（詳細はあとで説明します）。”connection”はコンポーネント間の接続を設定します。

以下は設定の例です:

GrxModelItem¶

**com.generalrobotix.ui.item.ModelItem**設定アイテムでは、以下の設定ができます。

ロボットモデルに関する設定¶

設定アイテム自身の”name”と”url”アトリビュートにはそれぞれ名前とモデルファイルの場所を設定します。

“isRobot”はモデルを動的パラメータを持ったロボットとして扱うことを設定します。

関節に関する設定¶

The name “*.mode” is used to specify the mode of the joint. If value “HighGain” is set, the joint can be controlled either by angle, velocity or acceleration of the joint. If not specified, the joint will be configured to use torque mode.

“*.angle”という名前の設定項目は、初期の関節角を設定します。

“*.velocity”という名前の設定項目は、初期の角速度を設定します。

“*.angularVelocity”という名前の設定項目は、初期の各運動速度を設定します。

リンクに関する設定¶

“*.translation”という設定項目は、[x, y, z]形式の初期のリンク位置を設定します。

“*.rotation”という設定項目はVRMLの角度形式の初期のリンク角度を設定します。

“*.collisionShape”という名前の設定項目は干渉チェックに用いられる方法を設定します。デフォルトでは精度の高い（が時間のかかる）アルゴリズムが用いられます。”convex hull”と設定されると、形状に凸包が適用されます。処理が高速化されますが凹みのあるモデルの干渉チェックの精度が低くなります。”AABB”と設定されると、干渉チェックにBounding boxが利用されます。

“*.NumOfAABB”はAABBの数を設定します。この数が少なくなると高速化されますが、精度は低くなります。

RTコンポーネントに関する設定¶

“rtcName”はモデルをRTコンポーネントとして操作する際の名前を設定します。

“inport”と”outport”はRTコンポーネントの入力ポートと出力ポートを作成します。値のフォーマットは”[port name]:[elements]:[data type]”です。

入力ポートと出力ポートには以下のデータ型を指定することができます:

JOINT_VALUE

element部で指定された関節の角度

JOINT_VELOCITY

element部で指定された関節の速度

JOINT_ACCELERATION

element部で指定された関節の加速度

JOINT_TORQUE

element部で指定された関節のトルク

ABS_TRANSFORM

element部で指定された関節の絶対座標

ABS_VELOCITY

element部で指定された関節の絶対速度

ABS_ACCELERATION

element部で指定された関節の絶対加速度

入力ポートには以下のデータ型を指定することができます:

FRAME_RATE

element部で指定された画像センサのフレームレート

LIGHT_SWITCH

element部で指定されたライトのオンオフスイッチ

出力ポートには以下のデータ型を指定することができます:

FORCE_SENSOR

力センサの値。

RATE_GYRO_SENSOR

レートジャイロセンサの値

ACCELERATION_SENSOR

加速度センサの値。

RANGE_SENSOR

レンジセンサの値。

VISION_SENSOR

画像センサの値。

POINT_CLOUD

ポイントクラウドの値。

GrxCollisionPairItem¶

GrxCollisionPairItemには以下の設定ができます:

objectName1

干渉チェックするオブジェクトの名前を設定します。

objectName2

干渉チェックする相手のオブジェクトの名前を設定します。

jointName1

干渉チェックするオブジェクトのリンクの名前を指定します。指定がない場合、そのオブジェクトのすべてのリンクがチェックされます。

jointName2

干渉チェックする相手のオブジェクトのリンクの名前を指定します。指定がない場合、そのオブジェクトのすべてのリンクがチェックされます。

slidingFriction

滑り摩擦係数を設定します。

staticFriction

静止摩擦係数を設定します。

cullingThresh

Cullingしきい値を設定する。

Restitution

反発係数を設定する。

GrxWorldStateItem¶

GrxWorldStateItemには以下のプロパティを設定することができます:

totalTime

シミュレーション時間を設定する。

timeStep

シミュレーションステップを設定する。

realTime

“true”の場合、実時間に同期する。

gravity

重力係数。

method

“EULER”の場合、オイラー法を使う。それ以外はルンゲクッタ法を使う。

integrate

“false”に設定された場合、kinematics onlyモードを使う。

GrxPythonScriptItem¶

GrxPythonScriptItemには以下のプロパティを設定することができます。

name

スクリプトアイテムの名前を設定する。

url

PythonスクリプトのURLを設定する。

Grx3DView¶

Grx3DViewItemには以下のプロパティを設定することができます:

showScale

シミュレーション画面に目盛を表示する。

showCoM

シミュレーション画面に重心を表示する。

showCoMonFloor

シミュレーション画面の床に重心を投影する。

showCollision

シミュレーション画面に接触点を表示する。

eyeHomePosition

シミュレーション画面の視野角を設定する。
